Explicação
pandas: Usado para manipulação e estruturação de dados (DataFrames).

numpy: Usado para operações numéricas e geração de dados sintéticos.

scikit-learn==1.3.0: A biblioteca principal de Machine Learning, usada para o Random Forest Classifier, StandardScaler, e StratifiedKFold. É crucial especificar a versão (1.3.0) para replicabilidade, conforme o Apêndice.

imbalanced-learn==0.11.0: Usada para lidar com o desequilíbrio de classes, especificamente para a implementação do SMOTE.
